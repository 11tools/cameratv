# AndroidTV USB Camera TV input 应用



通过AndoridTV的TIF（TV input framework）框架和LiveTV应用，把USB摄像头采集的图像作为输入源显示出来



## 使用方法

1. 在LiveTV上面选择本中间件

![2](.\figure\2.png)



2. 点击done后即开始Camera预览

   

![7](.\figure\7.png)

![8](.\figure\8.png)

![7](.\figure\7.png)



注意需要给中间件Camera权限，如果没有显示，在设置里面打开Camera权限

![10](.\figure\10.png)
